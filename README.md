# AnimeList

Simple Anime List API demo using React, Apollo GraphQl, CSS Modules and Material UI

API: [AniList](https://anilist.gitbook.io/anilist-apiv2-docs/)

Demo: https://animelist.now.sh/

## Setup

```bash
npm install
```
or
```bash
yarn
```
## Run

```bash
npm run start
```
or
```bash
yarn start
```
